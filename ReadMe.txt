All scripts are written in Python 3.6

USAGE:
Askisi3.py
ROT-13 Encryption script. Execute as-is. Input any (sane) text you please. Only latin characters are encrypted

Askisi4.py
Latin to Roman numerals converter. Execute as-is. Input any integer up to one million.

Askisi8.py
The script generates a list of 30 random integers from -30 to 30 and prints the triad combinations that add up to 0 (pressing 'n' terminates the script)
Execute as-is

Askisi11.py
Make any file with one word per line. Use the user created file as an argument when executing the script.
The script generates a 100x100 square of random latin characters and prints the words found in the square (vertically or horizontally)
